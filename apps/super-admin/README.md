# Super Admin Dashboard

This is the complete admin dashboard with full access to all Shribi Edufy features.

## Features

- Complete student management
- Staff management
- Fee management (all features)
- Transport management
- Inventory management
- Attendance tracking
- Homework & assignments
- Exams & results
- System settings
- User management

## Access

**Role Required:** `super_admin`

## Development

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Port

Development: `http://localhost:7000`

## Users

Super Admin users have access to:
- All modules
- System settings
- User management
- All reports and analytics
