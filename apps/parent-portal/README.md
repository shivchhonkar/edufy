# Parent Portal

Simple, mobile-friendly portal for parents to access their children's information.

## Features

- ✅ View children's information
- ✅ Check attendance
- ✅ View and pay fees
- ✅ See homework assignments
- ✅ View report cards
- ✅ Join online classes
- ✅ Download receipts and documents
- ✅ Get notifications

## Access

**Role Required:** `parent`

## Development

```bash
# Run development server
npm run dev

# Build for production
npm run build
```

## Port

Development: `http://localhost:3001`

## Design

- Mobile-first responsive design
- Simple, clean interface
- Large, easy-to-tap buttons
- Clear navigation
- Optimized for parents on-the-go
